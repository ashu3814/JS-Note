## ðŸ§  What is a Function?
A function is a block of code designed to perform a particular task. It runs only when called.

---

## ðŸ”¤ Syntax of a Function Declaration
```js
function functionName(parameters) {
  // code to execute
  return value; // optional
}
```

---

## ðŸ” Example: Simple Function
```js
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("John")); // Hello, John!
```

---

## ðŸ“ Local Variables
Variables declared inside a function are **local** to that function.
```js
function showMessage() {
  let message = "Hello, local variable!";
  console.log(message);
}

showMessage();
```

---

## ðŸŒ Outer Variables (Closure)
A function can access variables outside of it.
```js
let user = "Alice";

function greetUser() {
  console.log("Hello, " + user);
}

greetUser();
```

---

## ðŸ“¦ Parameters and Arguments
Functions can take parameters which behave like local variables.
```js
function sum(a, b) {
  return a + b;
}

console.log(sum(3, 4)); // 7
```

---

## ðŸŽ¯ Default Values
You can assign default values to parameters.
```js
function greet(name = "Guest") {
  console.log("Hello, " + name);
}

greet(); // Hello, Guest
greet("Sam"); // Hello, Sam
```

---

## ðŸ”™ Returning a Value
Functions can return a result using `return`.
```js
function multiply(a, b) {
  return a * b;
}

let result = multiply(2, 5);
console.log(result); // 10
```

---

## ðŸ· Naming a Function
Use camelCase. Function names should describe what they do.
Examples: `getUser()`, `calculateSum()`, `showMessage()`
